The Organization

I chose to work for The Wikimedia Foundation. I simply love The Wikimedia Foundation's idea of free knowledge sharing for one and all. Its an amazing community with very friendly people. I never felt I was new to this community.

The Project

I chose to work on jQuery.IME project. jQuery.IME is a jQuery based input method editor library supporting more than 155 input methods across more than 50 languages. It allows you to write in more than 50 languages using the default keyboard layout.

I chose this project because I have always had a fascination for Javascript and since this project was almost entirely written in Javascript, I knew I would love to hack around it. My project mainly aims at:

  • Porting jQuery.IME to browser extensions (Mozilla Firefox and Google Chrome)
  • Making some improvements in the library such as adding an on-screen keyboard feature and support for contenteditable divs
